TROUBLESHOOTING
================================================================================

If Ollama says "not found":
  1. Download: https://ollama.com/download
  2. Install it
  3. Open Command Prompt
  4. Type: ollama serve
  5. Keep that window open while using JARVIS

If test says "API not configured":
  1. Check .env file exists
  2. Check format: GROQ_API_KEY=gsk_xxxxx
  3. No extra spaces or quotes
  4. Save file
  5. Restart JARVIS

If JARVIS not responding:
  1. Check ollama serve is running
  2. Check internet connection
  3. Check data/jarvis.log for errors
  4. Run: python verify_installation.py
  5. Restart JARVIS

For more help:
  - Read: FREE_AI_SETUP.txt (troubleshooting section)
  - Check: data/jarvis.log (error details)
  - Run: python test_free_api_system.py (diagnostic info)
