4. IF WEATHER STOPS WORKING - TROUBLESHOOTING
================================================================================

ISSUE 1: "Weather API key not set"
  FIX:
    1. Get free key from: https://openweathermap.org/api
    2. Add to .env: WEATHER_API_KEY=your_key_here
    3. Restart JARVIS

ISSUE 2: "Weather service unavailable (HTTP 404)"
  FIX:
    1. Use city name, not state name
    2. Check spelling of city
    3. Try major city in that state

ISSUE 3: "No internet connection"
  FIX:
    1. Check your internet connection
    2. Verify firewall allows api.openweathermap.org
    3. Try pinging the API

ISSUE 4: Getting no response
  FIX:
    1. Check API key is valid (sign in to openweathermap.org)
    2. Check you haven't exceeded 1000 calls/day limit
    3. Try with a different city to verify API works
    4. Check API status: https://status.openweathermap.org
